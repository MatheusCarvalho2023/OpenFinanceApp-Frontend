Creates a BankList widget.

The banks parameter is required and specifies the list of banks to display.

The onBankSelected parameter is required and defines the callback that will be invoked when the user selects a bank from the list.

The optional selectedBankIndex parameter indicates which bank is currently selected. If null, no bank is visually marked as selected.
